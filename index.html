La Wild Code School

La Wild Code School est un réseau d’écoles présent actuellement dans 14 villes, en France et en Belgique. Ouverte à tous et sans prérequis de diplôme, elle qui propose des formations spécialisées. Les formations permettent d’accéder aux métiers numériques d’avenir : développeur web ou mobile, data analyst, product manager.

Cette page a pour objectif de vous présenter la promotion de JAVA J2EE du campus de Reims 2018-2019. Vous allez découvrir les profils des 12 développeur.se.s en formation.

Si vous souhaitez rejoindre la Wild Code School, il vous suffit de postuler sur le site et de participer à un Wild Breakfast. La prochaine rentrée est en février 2019 !
